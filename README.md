# web-portal-notes

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放与 web 门户相关的静态 HTML 页面。这些页面彼此独立，不针对任何特定域名或网站，可作为日常记录、学习笔记或临时页面的归档空间。

## 目录结构

```
web-portal-notes/
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 公共资源文件（CSS、JS、图片等）
└── README.md       # 项目说明
```

- `pages/`：每个 HTML 文件代表一个独立页面，文件名通常反映页面内容或用途。
- `assets/`：存放页面引用的外部资源，按类型或功能组织。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面内容以学习、测试或临时记录为主，不保证长期可用或更新。
- 归档遵循“一个文件一个页面”的原则，方便直接通过浏览器打开或部署到静态托管服务。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `pages/` 目录下的 HTML 文件查看内容。
3. 也可将整个 `pages/` 目录部署到任意静态文件服务器。
